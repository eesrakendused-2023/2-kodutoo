# 2nd homework

This is a simple webpage with a fixed navbar that allows users to navigate between different sections. It uses jQuery, GSAP, ScrollMagic, and Isotope libraries.

## Installation
To run this webpage, simply download all the files included in the repository and open the index.html file in your browser.

## Usage
Click on the links in the navbar to navigate between different sections. You can filter the portfolio section by clicking on the different categories in the filter button group.
